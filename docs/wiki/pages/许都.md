# 许都

## 概况
许都（今河南省许昌市）是[[曹操]]迎奉[[汉献帝]]后的临时都城。196年，曹操采纳[[荀彧]]的建议，将流亡的[[汉献帝]]迎至许县，改称许都。此后直到220年曹丕篡汉，许都一直是东汉朝廷的实际所在地，也是曹操政权的政治中心。

## 历史意义
许都成为都城的标志性意义在于曹操"挟天子以令诸侯"政治策略的成功实施。曹操以天子的名义征讨四方，获得政治上的合法性优势。许都期间，曹操在此推行屯田制，发展经济，招揽人才。[[郭嘉]]、[[荀彧]]、[[荀攸]]等大批谋士汇集许都。[[关羽]]曾在此斩[[颜良]]解白马之围。曹丕篡汉后迁都[[洛阳]]，许都仍为曹魏五都之一。

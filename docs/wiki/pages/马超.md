# 马超

**基本信息**
- **姓名**: 马超
- **字**: 孟起
- **别名**: 马孟起、西凉锦马超
- **生卒**: 176年 - 222年
- **势力**: [[蜀]]
- **身份**: 蜀汉五虎上将之一

## 性格特点
马超性格勇猛刚烈，武艺超群，有"不减吕布之勇"之称。他出身西凉名门，具有极强的军事才能和领导能力。性格相对单纯，对家族荣誉极为重视。作战勇猛，善于骑兵作战，是西凉军的代表性人物。

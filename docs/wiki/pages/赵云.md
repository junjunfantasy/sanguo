# 赵云

- **姓名**: 赵云

## 性格特点
赵云性格沉稳谨慎，忠勇双全，具有极强的责任感和使命感。他为人正直，不贪财不好色，深得[[刘备]]信任。作战勇猛，但处事谨慎，从不鲁莽。被称为"常胜将军"，一生征战鲜有败绩。

# 张飞

**基本信息**
- **姓名**: 张飞
- **字**: 翼德
- **别名**: 张翼德
- **生卒**: 168年 - 221年
- **势力**: [[蜀]]
- **身份**: 蜀汉五虎上将之一

## 性格特点
张飞性格豪爽直率，勇猛过人，但也暴躁易怒。他对[[刘备]]忠心耿耿，敬重士大夫（如对[[诸葛亮]]），但对部下严厉，经常鞭打士卒，最终因此被部下暗杀。武艺超群，有"万人敌"之称。

## 关键经历

### 早年经历
- **涿郡屠户**: 出身涿郡屠户
- **桃园结义**: 与[[刘备]]、[[关羽]]结为异姓兄弟
- **早期征战**: 跟随[[刘备]]四处征战

### 重要战役
- **虎牢关之战**: 在[[虎牢关]]与[[吕布]]大战
- **长坂坡之战**: 在长坂坡据水断桥，喝退[[曹军]]
- **入川之战**: 随[[刘备]]入川，立下战功
- **瓦口关之战**: 大败[[张郃]]

### 镇守阆中
- **镇守阆中**: [[刘备]]称帝后，张飞镇守[[阆中]]
- **准备伐吴**: [[关羽]]被杀后，准备随[[刘备]]伐吴
- **被部下暗杀**: 221年被部下[[范强]]、[[张达]]暗杀

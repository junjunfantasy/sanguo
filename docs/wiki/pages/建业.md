# 建业

## 概况
建业（今江苏省南京市）是[[东吴]]的都城，位于长江下游南岸。211年，[[孙权]]将治所从[[京口]]迁至秣陵，次年改名建业。建业背靠钟山，面临长江，地势险要，是控扼长江中下游的战略要地。孙权在此修建了石头城、太初宫等建筑，奠定了南京作为六朝古都的基础。

## 历史意义
建业作为东吴都城，见证了三国时期江东政权的兴起与衰落。从[[赤壁之战]]到[[夷陵之战]]，东吴以建业为中心，凭借长江天险与[[曹魏]]对抗。孙权在此称帝后，建业成为与[[洛阳]]、[[成都]]并立的三国都城之一。东吴大力发展江南经济，建业成为当时重要的商业和手工业中心。280年，西晋大军攻破建业，[[孙皓]]出降，东吴灭亡。

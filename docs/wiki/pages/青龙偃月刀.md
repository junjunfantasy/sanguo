# 青龙偃月刀

## 描述
青龙偃月刀是[[关羽]]的成名兵器，又名"冷艳锯"，重八十二斤（约合今41公斤）。刀身狭长呈偃月形，刀背刻有青龙纹饰，刀刃锋利无比。此刀在《三国演义》中被描绘为天下第一刀，是关羽威震华夏的标志。关羽骑[[赤兔马]]，持青龙偃月刀的形象被后世供奉为武财神和关帝圣君，成为忠义的象征。

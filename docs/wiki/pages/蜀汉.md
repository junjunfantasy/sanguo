# 蜀汉

## 概况
蜀汉（221年—263年）是三国时期刘备在西南建立的政权，以汉室正统自居，延续汉朝国祚。疆域涵盖今四川、重庆、云南、贵州及陕西南部。刘备以[[成都]]为都城，在[[诸葛亮]]辅佐下，联吴抗魏，形成了三足鼎立的局面。蜀汉以"汉"为国号，史称"蜀汉"或"季汉"。

## 历史沿革
**创业时期（184-214年）**：刘备以织席贩履起家，在[[关羽]]、[[张飞]]支持下参与镇压黄巾军。辗转依附[[公孙瓒]]、[[陶谦]]、[[曹操]]、[[袁绍]]等人，三顾茅庐请[[诸葛亮]]出山。借[[荆州]]为根基，西取[[益州]]，最终占据巴蜀。

**鼎盛时期（214-222年）**：刘备称汉中王后即帝位，国号汉。[[关羽]]北伐威震华夏，但东吴背盟夺取荆州。刘备为报仇发动[[夷陵之战]]，惨败于[[陆逊]]。

**维持时期（223-263年）**：[[诸葛亮]]辅佐后主[[刘禅]]，平定南中，五次北伐中原。[[姜维]]继其志，九伐中原，终因国力悬殊被[[曹魏]]所灭。

# 三国杀游戏整合

## 概述

三国杀游戏整合是将《三国志》历史数据与《三国杀》卡牌游戏数据进行双向映射与机制对齐的环节。三国杀卡牌涉及人物技能、武器装备、势力归属、人物关系等多个维度，这些都可以从构建的三国知识库中获取历史和语义支持。本流程定义从历史数据到游戏机制的映射规则、技能设计的知识支撑方法，以及游戏数据回写知识库的双向整合策略。

## 输入

- 三国知识图谱（实体、关系、事件数据）
- 《三国杀》卡牌数据集（标准版+风林火山+一将成名等扩展包）
  - 武将牌：人物名、势力、体力、技能描述
  - 装备牌：武器、防具、坐骑、宝物
  - 锦囊牌：策略/计谋类卡牌
- 三国杀规则集（基础规则、技能结算规则）
- 《三国杀》官方FAQ和社区卡牌数据库

## 处理流程

### 1. 人物-武将映射

- **核心映射**：将《三国杀》中的每个武将对应到知识库中唯一的人物实体
  - 通过别名匹配（"曹操"="曹孟德"），辅以技能特征验证
  - 同名但不同版本的武将（如界曹操、SP曹操）映射到同一个人物实体，标记版本差异
- **势力映射**：三国杀势力（魏/蜀/吴/群/神）对应历史势力归属
  - 验证：某人物在历史上的势力归属与卡牌势力是否一致
  - 争议处理：吕布（群雄vs袁术部将）、张鲁（群雄vs汉中独立势力）
- **体力映射**：三国杀体力值对应人物在历史上的存活能力
  - 历史参考：死亡年份-出生年份推算寿命，长命者体力高
  - 调整因子：勇猛武将体力+1，文官体力-1

### 2. 技能-历史关联映射

- **技能溯源**：每个技能名称/效果从历史事件或人物特征中溯源
  - 案例1：曹操技能"奸雄"←《三国志》"太祖少机警，有权数"→对应"奸雄"特质
  - 案例2：诸葛亮技能"观星"←《三国演义》中诸葛亮的观星天文能力
  - 案例3：关羽技能"武圣"←后世对关羽的武圣封号
- **技能分类**：按历史支撑类型分类
  | 技能类型 | 历史来源 | 示例 |
  |---------|---------|------|
  | 事迹型 | 具体历史事件 | 赵云"龙胆"（长坂坡） |
  | 特质型 | 人物性格/能力 | 司马懿"忍戒" |
  | 封号型 | 谥号/封号/称号 | 关羽"武圣"、华佗"神医" |
  | 虚构型 | 《三国演义》虚构 | 诸葛亮"空城" |
  | 衍生型 | 后世演绎/文化 | 貂蝉"离间" |

### 3. 装备-历史物品映射

- **武器映射**：青龙偃月刀→关羽、丈八蛇矛→张飞、方天画戟→吕布
  - 从知识库的Artifact节点查找武器归属人物
  - 武器效果设计参考武器的历史特征（如"贯石斧"的强力破防对应程咬金的勇猛）
- **防具坐骑**：赤兔马（吕布→关羽）、爪黄飞电（曹操）
  - 追查名马名甲的历史归属变迁
- **宝物映射**：玉玺→传国玉玺（孙坚/袁术）、木牛流马→诸葛亮发明

### 4. 游戏机制与历史对齐

- **身份局机制映射**：
  - 主公→历史中的势力首领（曹操、刘备、孙权）
  - 忠臣→该势力的主要谋士/武将（荀彧、诸葛亮、周瑜）
  - 反贼→敌对势力人物（司马懿对曹魏而言是反贼）
  - 内奸→野心家/反复之徒（吕布、孟获）
- **锦囊牌设计**：基于历史事件/策略设计
  - "无中生有"←三十六计"无中生有"
  - "借刀杀人"←借孙权之手杀关羽
  - "南蛮入侵"←诸葛亮南征
  - "万箭齐发"←赤壁之战箭雨

### 5. 双向数据整合

- **历史→游戏**：知识库为卡牌设计提供历史依据和背景故事素材
- **游戏→历史**：游戏中的技能效果数据回写知识库，标注为"游戏衍生属性"
- **新武将设计建议**：从知识库中寻找尚未被三国杀覆盖的历史人物
  - 条件：在知识库中Faction属性明确、有可提取的技能特征
  - 建议输出：技能名、效果描述、历史依据、引用出处

## 输出

- 人物-武将映射表（知识库ID↔三国杀武将ID，含版本信息）
- 技能-历史事件关联表（每个技能对应的历史来源及置信度）
- 装备/锦囊-历史物品/策略关联表
- 新武将设计建议列表（基于知识库未被覆盖的人物）
- 游戏机制合理性分析报告

## 常见问题

1. **演义vs史实冲突**：三国杀大量引用《三国演义》虚构设定，需与正史分开标注
2. **技能平衡性**：历史重要人物在游戏中不一定强力（如刘禅），需理解游戏平衡性优先于历史还原
3. **势力归属**：部分人物一生效力多个势力（如张辽：吕布→曹操），卡牌取最终归属
4. **版本演进**：三国杀不断推出新扩展包，知识库需持续更新同步

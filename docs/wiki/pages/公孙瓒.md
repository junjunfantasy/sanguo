# 公孙瓒

## 概况
公孙瓒（？—199年）是东汉末年割据幽州的军阀，以骑兵精锐闻名，麾下有"白马义从"骑兵部队，被胡人称为"白马将军"。他曾长期与北方游牧民族作战，威震塞外。后与[[袁绍]]争夺河北霸权，初期占据优势，但因刚愎自用、滥杀士人，最终在[[易京之战]]中兵败自焚而死。

## 历史沿革
**崛起时期（189-193年）**：公孙瓒出身贵族但家道中落，从郡中小吏做起，因作战勇敢升为辽东属国长史。他组建白马义从骑兵，屡破乌桓、鲜卑，威名远扬。在讨伐[[董卓]]的关东联军中，公孙瓒是重要力量。

**争霸时期（193-196年）**：公孙瓒与[[袁绍]]争夺河北霸权，初期在[[界桥之战]]中与袁绍主力激战，但因战术失误而败。此后与袁绍拉锯战于河北，互有胜负。公孙瓒残暴好杀，迫害士族[[刘虞]]，导致幽州人心尽失。

**覆灭时期（196-199年）**：公孙瓒被袁绍步步紧逼，退守[[易京]]，筑高楼自保。他坐视部下被围而不救，军心尽失。[[袁绍]]围攻[[易京]]，公孙瓒派人向黑山军求救。后误以为救援已至，突围失败，自焚而死。

# 司马懿

**基本信息**
- **姓名**: 司马懿
- **字**: 仲达
- **别名**: 司马仲达
- **生卒**: 179年 - 251年
- **势力**: [[魏]]
- **身份**: 曹魏重臣，晋朝奠基人

## 性格特点
司马懿性格深沉隐忍，城府极深，具有极强的政治洞察力和忍耐力。他善于隐藏自己的真实意图，"狼顾之相"暗示其野心。性格谨慎，从不轻易冒险，善于等待时机。具有卓越的军事才能和政治手腕，最终为司马氏篡魏奠定基础。

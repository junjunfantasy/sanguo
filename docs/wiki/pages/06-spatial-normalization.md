# 空间归一化

## 概述

空间归一化是将古籍中的古代地名映射到现代地理坐标，并处理三国时期复杂的行政区划变迁。三国时期（220-280年）行政区划经历了东汉十三州→魏蜀吴三国分治的剧烈变动，地名重叠、同名异地、异地同名现象突出。本流程建立古代地名→现代地名的多级映射体系，同时为每个地名标注时间范围，实现地名溯源。

## 输入

- 实体提取中的地名实体（E02地点、E17行政区划、E16城防、E18山水）
- 事件抽取中的事件地点信息
- 参考地图：谭其骧《中国历史地图集》第三册（三国时期）
- 参考辞典：《中国历史地名大辞典》《三国志地名汇释》

## 处理流程

### 1. 地名类型分类

| 类型 | 覆盖范围 | 示例 | 映射难度 |
|------|---------|------|---------|
| 州级 | 东汉十三州 | 荆州、益州、豫州 | 中（区域变化大） |
| 郡级 | 三国时期约160郡 | 颍川郡、南阳郡、琅邪郡 | 高（治所多变） |
| 县级 | 三国时期约1200县 | 许县、谯县、阳翟县 | 高（大量已废） |
| 城池 | 具体城池关隘 | 许都、邺城、潼关 | 中（位置较明确） |
| 山川河流 | 山名/水名 | 赤壁、官渡、街亭 | 高（位置具争议） |
| 特殊地名 | 要塞/陵墓/宫殿 | 铜雀台、五丈原、阿房宫 | 高（部分已毁） |

### 2. 古今地名映射

- **三级映射**：古代地名→现代行政区划→经纬度坐标
- **映射表结构**：{古名, 朝代, 起始时间, 结束时间, 古政区级别, 今省, 今市, 今区县, 纬度, 经度, 置信度, 来源}
- **核心映射策略**：
  - 古籍地名→谭其骧地图位置→国家测绘局行政区划代码→经纬度
  - 对迁移型地名（如"荆州"治所多次迁移），按时间段分别映射
- **同名消歧**：同名异地（如"东郡"东汉属兖州，三国时属曹魏→今河南濮阳）
- **地名演变链**：记录地名历史演变序列（如"许县→许都→许昌→魏都区"）

### 3. 行政区划变迁处理

- **东汉末（184-220年）**：十三州制，州-郡-县三级，参考《后汉书·郡国志》
- **三国鼎立（220-265年）**：
  - 曹魏：司隶+12州（新增凉州、雍州）
  - 蜀汉：益州（含南中七郡）
  - 东吴：扬+荆+交+广4州
- **西晋初（265-280年）**：统一后十九州
- **变迁标注**：每个地名标注"有效时间窗"，关联政区变更事件

### 4. 争议地点处理

| 争议地名 | 各方案 | 采用策略 |
|---------|-------|---------|
| 赤壁 | 蒲圻说/黄冈说/武昌说 | 采用主流蒲圻说（今赤壁市），标注备选 |
| 街亭 | 甘肃秦安/庄浪 | 秦安陇城镇为主，庄浪韩店为辅 |
| 官渡 | 中牟说/原阳说 | 中牟县官渡镇为主方案 |
| 祁山 | 礼县/西和/天水 | 礼县祁山镇为主 |

## 输出

- 地名映射数据库（JSON/CSV）：古代地名→现代坐标+时间窗+置信度
- 地名变迁时间线：每个地名的政区变化时序
- 争议地点报告：标注争议方案及参考文献
- KML/GEOJSON文件：可导入Google Earth/QGIS可视化

## 常见问题

1. **治所迁移**：同一州/郡在不同时期治所不同（如荆州治所:武陵→襄阳→江陵），需按时间分段
2. **地名消失**：大量三国县名已废（如"阴沟县"），标注为"已废-今预估区域"
3. **边界变动**：郡县边界古今变迁巨大，标注"概略位置"而非精确边界
4. **自然地名演变**：河道变迁（黄河改道），水战地点需考虑不同时代水文

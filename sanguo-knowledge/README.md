# 三国知识库（sanguo-knowledge）

这是一个关于三国时期（东汉末年至西晋统一）的综合性知识库，
融合了《三国志》正史、《三国演义》文学和三国杀游戏数据。

## 项目结构

- `characters/` — 人物知识（41人）
- `events/` — 历史事件
- `battles/` — 重大战役
- `factions/` — 势力
- `locations/` — 地点
- `weapons/` — 武器与宝物
- `skills/` — 三国杀技能
- `strategies/` — 计策策略
- `concepts/` — 典故成语
- `books/` — 文献典籍

## 如何使用

1. **在 Obsidian 中**：将此文件夹作为仓库打开
2. **浏览**：使用 `[[]]` 双向链接导航
3. **图谱**：打开 Obsidian 图谱视图查看所有关联
4. **搜索**：使用全局搜索查找人物/事件/地点

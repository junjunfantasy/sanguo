# 时间归一化

## 概述

时间归一化是将三国原著中使用的多种纪年方式统一转换为公元纪年，建立时间维度的统一坐标。三国时期（184-280年）涉及东汉末、三国鼎立、西晋初三个时期的年号混用，且魏、蜀、吴各有独立纪年系统，时间处理复杂度较高。本流程覆盖年号→公元转换、干支纪年解析、模糊时间处理、时间争议仲裁四大任务。

## 输入

- 事件抽取结果中的原始时间表达（年号年、干支、相对时间、模糊时间）
- 三国年号对照表（东汉/曹魏/蜀汉/东吴四系统）
- 陈垣《二十史朔闰表》、方诗铭《中国历史纪年表》
- 天文历法数据（用于干支精校）

## 处理流程

### 1. 年号→公元纪年转换

- **年号-年份映射表**：构建四套年号对照表
  - 东汉系统：建宁·168-172、熹平·172-178、光和·178-184、中平·184-189、初平·190-193、兴平·194-195、建安·196-220、延康·220
  - 曹魏系统：黄初·220-226、太和·227-233、青龙·233-237、景初·237-239、正始·240-249、嘉平·249-254、正元·254-256、甘露·256-260、景元·260-264、咸熙·264-265
  - 蜀汉系统：章武·221-223、建兴·223-237、延熙·238-257、景耀·258-263、炎兴·263
  - 东吴系统：黄武·222-229、黄龙·229-231、嘉禾·232-238、赤乌·238-251、太元·251-252、神凤·252、建兴·252-253、五凤·254-256、太平·256-258、永安·258-264、元兴·264-265、甘露·265-266、宝鼎·266-269、建衡·269-271、凤凰·272-274、天册·275-276、天玺·276、天纪·277-280
- **解析规则**："年号+数字+年"模式（如"建安五年"→200年），"年号+初/末/中"等模糊模式处理
- **跨系统转换**：同一事件可能被不同纪年系统记载，需按实际记载朝代转换

### 2. 干支纪年处理

- **干支-公元映射**：60年周期循环，以184年甲子为基准点
- **自动推算**：公式：公元年=(干支序数-基准干支序数) mod 60 + 基准年
- **防错机制**：推算结果与上下文事件时间比对，若偏差>30年则标记复查
- **月/日干支**：月干支结合节气判定，日干支需查朔闰表

### 3. 模糊时间处理

| 原文表达 | 处理规则 | 示例转换 |
|---------|---------|---------|
| 初（年号） | 年号起始年 | "初平初"→190年 |
| 年号+末 | 年号最后一年 | "建安末"→220年 |
| 年号+中 | 年号中间年份 | "建安中"→208年左右 |
| 明年 | 事件上文中最近年份+1 | - |
| 顷之 | 同上文事件同年 | - |
| 是岁 | 同上文事件同年 | - |
| 春/夏/秋/冬 | 补充为3/6/9/12月 | "建安五年春"→200年3月 |
| 暮春/仲夏等 | 细化到月份 | "仲夏"→6月 |

### 4. 时间争议处理

- **《三国志》vs《后汉书》**：同一事件记载时间不同时，优先采用《三国志》本纪
- **魏蜀吴三国纪年差异**：同一事件在三国记载中年号不同，以事件实际发生地的纪年为准
- **裴注时间补充**：裴松之注中他书引文的时间信息作为补充证据
- **月日缺失**：仅标注年份，月日标记为00-00
- **争议标记**：概率最高的方案为主答案，其他可能时间为备选，附争议说明

## 输出

- 统一时间标注文件：每条时间表达→ISO 8601格式（YYYY-MM-DD），含置信度
- 年号→公元转换映射表（含所有年号年份的完整映射）
- 时间争议报告：标记所有有争议的时间记载及各来源对比

## 常见问题

1. **年号改元**：一年内多次改元（如220年延康→黄初），需精确到月
2. **跨年号事件**：事件跨年号持续（如赤壁之战跨年），标记为起始-结束
3. **蜀汉纪年缺失**：蜀汉后期记载简略，部分事件时间需从魏/吴记载反推
4. **朔闰误差**：古籍中月日记载的朔闰误差，使用天文软件（Stellarium）校验
